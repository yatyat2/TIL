# 비구조화 할당

props를 통해 값을 넘기는 방법외에 props를 나눠서 값을 넘기는 방법,

value와 name을 넘긴다고 할 때

const test(props) => {

props.value

props.name

}



const test(value,name) => {

value

name

}

이런식으로 넘길 수 있음



함수가 아니라 컴포넌트로 값을 넘길 때는

const {value,name} = this.props; 이렇게 해서 this.props.value, this.props.name 같이 길게 쓰는일을 방지할 수 있음
