# HOW TO USE

### Setup

설치는 이렇게

``` 
npm install --save next react react-dom
```



그리고 package.json에 아래의 스크립트를 추가하세요

```javascript
{
  "scripts": {
    "dev": "next",
    "build": "next build",
    "start": "next start"
  }
}

```



위와같이 하면 파일시스템이 메인 API가 됩니다. 모든 `.js` 파일은 자동으로 처리, 렌더링하는 루트가 됩니다.(번역 다시 해보기)
